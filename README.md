# Установка
* Скачиваем архив
* Распаковываем в удобную нам директорию
* Запускаем Terminal
* Переходим в корень с программой
* Запускаем установку доп. компонентов: python Install.py

# Запуск
* Вводим в Terminal: python "HZF Weather in your city V2.0.py"
